# SecureBunker

    Password and information manager    
## Description

    A simple program to keep your confidential information under control.
    Access to multiple platforms to save information anywhere. Local access, via ftp, google drive, etc.

    
## Dependencies

    NetCore 9.0
	Backend app: https://github.com/martisdev/NetCoreFileAccess	
	To access different data sources (local, ftp, googleDrive)
